Format:

Search for any type of publishing format that includes your search words

Language:

Select the Language you want results to be in (Selecting “French” will cause only french-language works to be displayed).

Location:

Restrict the search based on where the media is housed (What Library a book is at, for example)

Year:

Select a range of publishing dates to restrict search results to (Selecting 2000-2009 will only provide results from that decade, for instance).

Search:

Press this button to perform search in the library database based on the fields specified by the user.

Clear:

Pressing clear will empty all search fields and reset all drop-down menus. An easy way to reset the search!
